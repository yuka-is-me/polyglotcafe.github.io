安装基本组件

  1. 官网下载Python最新版本

  2. 官网下载Pycharm

Pycharm是一款IDE软件，即集成开发环境。

IDE: Integrated Development Environment

Pycharm作为Python语言的编译器，起到interpret或

translate的作用。将Python语言转化成计算机可以理解的二

进制语言，从而起到控制计算机进行操作的作用。

Python语言的特点

Python是一门对大小写非常敏感的语言，所以在进行coding

的时候，需要注意区分大小写。
Case sensitive language: sensitive to lower or upper class

characters.

Python文件的后缀是“.py”
